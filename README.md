# Customer-Shopping-Behavior-Analysis
Overview

This project analyzes customer shopping behavior using transactional data across multiple product categories. The goal is to understand customer demographics, spending patterns, and product preferences to support data-driven business decisions.

The analysis was carried out using Python (Pandas) for data cleaning and EDA, MySQL for business queries, and Power BI for interactive dashboard visualization.

Business Problem

Businesses often struggle to understand:

- Which products and categories drive the most revenue
- How customer segments behave and contribute to sales
- Whether discounts and subscriptions actually improve revenue
- How factors like seasonality, shipping, and demographics impact purchasing behavior

This project aims to solve these challenges by analyzing customer data to uncover actionable insights for improving sales, marketing strategies, and customer retention.

Key Performance Indicators (KPIs)

The following KPIs were used to evaluate business performance:

- Total Revenue – Total sales generated from all transactions
- Total Orders – Number of purchases made
- Average Purchase Value – Average spending per transaction
- Customer Count – Total number of customers

Project Description

This is an end-to-end data analysis project that combines data preprocessing, SQL analysis, and dashboard visualization.

- Performed data cleaning and preprocessing by handling missing values, standardizing column names, and creating new features such as age groups and purchase frequency.
- Conducted exploratory data analysis (EDA) to identify patterns in customer behavior and spending.
- Used MySQL to write business queries addressing key questions related to revenue, product performance, and customer segmentation.
- Built an interactive Power BI dashboard to visualize insights related to sales trends, customer demographics, and purchasing behavior.

The project provides actionable insights that can help businesses improve marketing strategies, customer experience, and revenue growth.
